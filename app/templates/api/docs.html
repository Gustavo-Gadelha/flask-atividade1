Uso da API

Acesse a página Token API ou envie suas credenciais para POST /api/token para obter um token JWT

  • Inserir Produto - POST /api/products

    Use este endpoint para adicionar um novo produto

  • Buscar Produto - GET /api/products/{id} ou GET /api/products/{nome}

    Use este endpoint para buscar um produto por ID ou nome

  • Listar Produtos - GET /api/products

    Use este endpoint para obter uma lista de todos os produtos

  • Listar Usuários - GET /api/users

    Use este endpoint para obter uma lista de todos os usuários
